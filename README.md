# printf

Viewer for FreeRTOS profiles produced by [scanf](https://github.com/try-fork-it-twice/scanf)


## Development

**Prerequisites:**

- Python 3.13
- Poetry

**Setup project:**

```shell
poetry install
```

**Run tests:**

```shell
poetry run pytest -v
```

**Type check with mypy:**
```shell
poetry run mypy src tests
```

**Format code:**

```shell
poetry run ruff format src tests
```
